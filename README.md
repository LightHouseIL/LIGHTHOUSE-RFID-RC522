# LIGHTHOUSE RFID for MakeCode MFRC522.

## Connection

יש לחבר את המודול על פי הרישום הבא:
| RFID module   | Micro:bit     |
| ------------- |:-------------:|
| VCC           | 3V            |
| GND           | GND           |
| MISO          | P14           |
| MOSI          | P15           |
| SCK           | P13           |
| NSS           | P16           |
